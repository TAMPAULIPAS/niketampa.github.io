# рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў  
рЃћрЃА рЃљрЃарЃўрЃА рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ рЃерЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.  

## ­ЪЊї рЃљрЃдрЃгрЃћрЃарЃљ  
рЃљрЃЦ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃгрЃћрЃарЃЮ, рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ.
